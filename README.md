## üìñ About

**Quiz CD5** is a web application designed for football fans to test their knowledge. The goal was to recreate the fun of online football quizzes while managing complex backend logic, such as text-based answer validation, user sessions, and administrative dashboards.

### Key Features
* **Authentication System:** User registration, login, and profile management.
* **Quiz Engine:** Free-text response fields with instant validation.
* **User Dashboard:** Progress tracking and historical results.
* **Admin Interface:** Full CRUD management for quizzes and questions.
* **Responsive Design:** Clean and simple UI built with Bootstrap.
